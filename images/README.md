# Architecture Images

This directory contains architecture diagrams and visual assets for the project.

## Files
- `architecture.png` - Main system architecture diagram showing the flow from Users → AWS Amplify → AWS Lambda → AWS AppSync → GraphQL → Amazon DynamoDB with Amazon Cognito for authentication.

To add the architecture diagram:
1. Save your architecture image as `architecture.png` in this directory
2. The README.md will automatically display it in the Architecture section
